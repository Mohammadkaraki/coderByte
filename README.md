# Article Search with Text Highlighting

A search application with real-time text highlighting feature.

## Steps to Run the Application

1. Clone the repository:
```bash
git clone https://github.com/Mohammadkaraki/coderByte.git
cd coderByte
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to:
```
http://localhost:3000
```
